Authentication

Python example of Authentication:

import requests

api_key = "XXXXXXXXXXXXXXXXXX"

url = 'http://127.0.0.1:5000/authenticate'
my_obj = {'api_key': f'{api_key}'}

request = requests.post(url, json=my_obj)

# Below is used to create the header for future requests
access_token = request.json()
access_token = access_token['access_token']
headers = {
    "Authorization": f"Bearer {access_token}"
}

Note: Account is required for an API Key
Note: the API Key can be found by navigating to "Get API Key" tab as shown below.
